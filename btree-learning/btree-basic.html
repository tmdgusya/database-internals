핵심 개념 (Key Concepts)

🎯 N Keys, N+1 Pointers

B-Tree 노드는 N개의 키를 가지면 N+1개의 포인터를 가집니다. 각 키는 자식 노드들을 구분하는 경계값 역할을 합니다.

📊 Range-based Search

각 포인터 Pᵢ는 특정 범위의 값들을 담고 있는 자식 노드를 가리킵니다. Kᵢ₋₁ ≤ Kₛ < Kᵢ 범위에 해당하는 포인터를 따라갑니다.

⚡ Implicit K₀ = -∞

첫 번째 포인터 P₀는 가장 작은 값들을 담고 있는 자식을 가리킵니다. 이는 K₀ = -∞로 간주되어 -∞ ≤ Kₛ < K₁ 범위에 해당합니다.

🌳 Balanced Tree

B-Tree는 모든 리프 노드가 같은 깊이에 있는 균형 잡힌 트리입니다. 이는 검색, 삽입, 삭제 연산이 O(log N) 시간 복잡도를 보장합니다.
